🎨 Breadboard Layout Guide

Power Rails Setup: • Connect NodeMCU 3V3 to positive power rail (red stripe) • Connect NodeMCU GND to negative power rail (blue/black stripe) Component Placement (Left to Right): Row 1-5: DHT11 Temperature & Humidity Sensor Row 10-15: MQ-2 Gas Sensor Row 20-25: Sound Detection Sensor Row 30-35: Traffic Light LEDs (Red, Yellow, Green) Row 40-45: Buzzer Row 50-55: Current Limiting Resistors (220Ω x 3)

⚡ Pin Configuration Summary

Input Sensors

D7 (GPIO13) - DHT11 Data Pin

D6 (GPIO12) - Gas Sensor Digital Out

D5 (GPIO14) - Sound Sensor Digital Out

Output Devices

D0 (GPIO16) - Red LED

D1 (GPIO5) - Yellow LED

D2 (GPIO4) - Green LED

D4 (GPIO2) - Buzzer

⚠️ Important Notes:
• Use 220Ω resistors for each LED to prevent burnout
• Ensure proper polarity for LEDs (longer leg = positive)
• DHT11 may need a 10kΩ pull-up resistor on data line
• Double-check all connections before powering on
• Gas sensor needs 24-48 hours warm-up time for accurate readings

🔧 Assembly Steps

  1. Power Setup: Connect 3V3 and GND rails on breadboard
  2. Sensors: Place DHT11, Gas sensor, and Sound sensor
  3. LEDs: Insert LEDs with 220Ω resistors in series
  4. Buzzer: Connect buzzer with proper polarity
  5. Wiring: Use jumper wires as per the table above
  6. Testing: Upload code and test each component individually
